# electronic-library

Веб-приложение **«Электронная библиотека»**.

## Автор

Тихонкова Анастасия, группа 241-3211

## Описание проекта

Веб-приложение предназначено для ведения электронного каталога книг. Система позволяет просматривать список книг, искать произведения по названию, автору, издательству и жанрам, открывать подробную страницу книги, оставлять рецензии и учитывать статистику просмотров.

В приложении реализовано разграничение прав доступа по ролям. Администратор управляет каталогом книг и просматривает статистику, модератор редактирует сведения о книгах, а пользователи могут просматривать каталог и оставлять рецензии.

## Основные возможности

* просмотр каталога книг;
* поиск по названию, автору, издательству и жанрам;
* просмотр подробной страницы книги;
* авторизация пользователей;
* разграничение прав доступа по ролям;
* добавление, редактирование и удаление книг;
* загрузка и хранение обложек;
* добавление рецензий и оценок;
* автоматический расчёт среднего рейтинга книги;
* отображение популярных книг;
* отображение недавно просмотренных книг;
* журнал просмотров;
* статистика просмотров книг;
* фильтрация статистики по датам;
* экспорт данных в CSV.

## Роли пользователей

* **Гость** — просматривает каталог и страницы книг.
* **Пользователь** — просматривает книги и оставляет рецензии.
* **Модератор** — редактирует информацию о книгах.
* **Администратор** — управляет книгами, удаляет записи и просматривает статистику.
