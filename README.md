# electronic-library

Веб-приложение на тему: **«Электронная библиотека»**.

## Автор

Тихонкова Анастасия 241-3211

## Описание проекта

Веб-приложение предназначено для ведения электронного каталога книг. Система позволяет просматривать список книг, искать произведения по названию, автору, издательству и жанрам, просматривать подробную информацию о книге, оставлять рецензии, а также учитывать статистику просмотров.

Приложение поддерживает разграничение прав доступа по ролям. Администратор может управлять каталогом книг и просматривать статистику, модератор может редактировать сведения о книгах, а пользователи могут оставлять рецензии.

## Основные возможности

* авторизация пользователей;
* разграничение прав доступа по ролям;
* просмотр каталога книг;
* пагинация списка книг;
* поиск книг без учёта регистра по названию, автору, издательству и жанрам;
* просмотр подробной информации о книге;
* отображение жанров, года издания, издательства, автора и объёма книги;
* загрузка и хранение обложек книг;
* хранение информации об обложках в отдельной таблице;
* вычисление MD5-хэша изображения обложки;
* добавление, редактирование и удаление книг;
* поддержка связи книг и жанров «многие ко многим»;
* написание рецензий к книгам;
* запрет повторной рецензии одним пользователем на одну книгу;
* отображение средней оценки и количества рецензий;
* поддержка Markdown в описаниях книг и текстах рецензий;
* очистка пользовательского Markdown-текста с помощью Bleach;
* отображение популярных книг за последние 3 месяца;
* отображение недавно просмотренных книг;
* учёт просмотров страниц книг;
* ограничение количества учитываемых просмотров одной книги одним пользователем в день;
* просмотр журнала посещений;
* просмотр статистики просмотров книг;
* фильтрация статистики по датам;
* экспорт журнала просмотров и статистики в формате CSV.

## Роли пользователей

В системе предусмотрены три основные роли:

* **Администратор** — имеет полный доступ к системе, может добавлять, редактировать и удалять книги, а также просматривать статистику;
* **Модератор** — может редактировать данные книг;
* **Пользователь** — может просматривать книги и оставлять рецензии.

Неаутентифицированные пользователи могут просматривать главную страницу, страницу входа и страницы с информацией о книгах.

## База данных

В проекте используется база данных SQLite:

```text
instance/library.sqlite
```

База данных создаётся автоматически при первом запуске приложения. В неё добавляются роли пользователей, тестовые пользователи, жанры, книги зарубежной литературы, обложки, рецензии и история просмотров.

Файл базы данных не требуется добавлять в репозиторий. Если удалить файл `instance/library.sqlite`, то при следующем запуске приложение заново создаст базу данных и заполнит её начальными данными.

Для повторного создания базы данных можно выполнить:

```powershell
Remove-Item instance\library.sqlite -ErrorAction SilentlyContinue
```

После этого нужно запустить приложение заново.

## Запуск проекта

1. Клонировать репозиторий:

```bash
git clone https://github.com/hellostacia/electronic-library.git
```

2. Перейти в папку проекта:

```bash
cd electronic-library
```

3. Создать виртуальное окружение:

```bash
py -m venv ve
```

4. Активировать виртуальное окружение:

```bash
ve\Scripts\activate
```

5. Установить зависимости:

```bash
pip install -r requirements.txt
```

6. Запустить приложение:

```bash
python app.py
```

7. Открыть приложение в браузере:

```text
http://127.0.0.1:5000
```

## Демонстрационный сценарий

Для проверки работы приложения можно выполнить следующий сценарий:

1. Открыть главную страницу и просмотреть каталог книг.
2. Воспользоваться поиском по названию, автору, издательству или жанру.
3. Открыть страницу выбранной книги.
4. Просмотреть описание книги, обложку, жанры, рейтинг и рецензии.
5. Войти под учетной записью пользователя.
6. Оставить рецензию к книге.
7. Войти под учетной записью модератора.
8. Отредактировать сведения о книге.
9. Войти под учетной записью администратора.
10. Добавить новую книгу с обложкой.
11. Удалить книгу через модальное окно подтверждения.
12. Открыть страницу статистики.
13. Проверить журнал просмотров книг.
14. Отфильтровать статистику просмотров по датам.
15. Скачать CSV-файлы с журналом просмотров и статистикой.
